## Your Profile Page

Every user has a public profile page at:
```
https://rtfm.guide/profile/{username}
```

## What's On Your Profile

### Profile Information

- Your name
- Join date
- Biography (coming soon)
- Location (coming soon)
- Website (coming soon)

### Your Guides

All your published guides are displayed on your profile:
- Title and description
- Publication date
- View count
- Reaction count

<Info>
  **Privacy**: Only published guides appear on your profile. Drafts and unlisted guides remain private.
</Info>

### Statistics (Coming Soon)

Your profile will show:
- Total guides published
- Total views across all guides
- Total reactions received
- Total comments
- Member since date
- Reputation score

## Viewing Other Profiles

Click any username throughout the site to view their profile:
- See their published guides
- View their contributions
- Follow them (coming soon)

## Editing Your Profile

Access profile settings at [https://rtfm.guide/settings/profile](https://rtfm.guide/settings/profile)

### Update Information

- **Name**: Change your display name
- **Email**: Update your email address
- **Biography**: Write a short bio (coming soon)
- **Social links**: Add GitHub, Twitter, etc. (coming soon)

### Profile Picture (Coming Soon)

- Upload a custom avatar
- Or use Gravatar
- Set default avatar style

## Privacy Settings (Coming Soon)

Control what others can see:
- Hide your email address
- Hide guides from profile
- Private profile option
- Activity visibility

## Profile URL

Your profile URL is based on your user ID by default:
```
https://rtfm.guide/profile/123
```

Custom usernames coming soon:
```
https://rtfm.guide/@yourusername
```

## Best Practices

### Complete Your Profile

A complete profile helps:
- Build trust with readers
- Get more engagement
- Connect with the community
- Showcase your expertise

### Professional Presence

- Use a real name or consistent handle
- Write a clear bio
- Add relevant social links
- Keep your guides updated
